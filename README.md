# README
Required
Ruby version 2.5.1
Rails version 5.2.4, >= 5.2.4.1

## Install
`git clone git@github.com/clelantoson/airbn_dogs.git`
`cd airbn_dogs`
`bundle install`
`rails db:reset`
`rails db:migrate`
`rails db:seed`
`rails c`
## Run (with gem 'table_print')

`tp Dog.all`
`tp Dogsitter.all`
`tp Stroll.all`
`tp City.all`
